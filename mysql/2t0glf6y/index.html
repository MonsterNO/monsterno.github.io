创建数据库

使用 create 命令创建数据库：

CREATE DATABASE [IF NOT EXISTS] 数据库名
    [CHARACTER SET 字符集]
    [COLLATE 排序规则];

如果你希望在创建数据库时指定一些选项，可以使用 CREATE DATABASE 语句的其他参数，例如，你可以指定字符集和排序规则：

CREATE DATABASE test
    CHARACTER SET utf8mb4
    COLLATE utf8mb4_general_ci;

如果数据库已经存在，执行 CREATE DATABASE 将导致错误。

为了避免这种情况，你可以在 CREATE DATABASE 语句中添加 IF NOT EXISTS 子句：

CREATE DATABASE IF NOT EXISTS test
    CHARACTER SET utf8mb4
    COLLATE utf8mb4_general_ci;

删除数据库

在删除数据库过程中，务必要十分谨慎，因为在执行删除命令后，所有数据将会消失。

使用 drop 命令删除数据库：

DROP DATABASE 数据库名;
DROP DATABASE IF EXISTS 数据库名;

选择数据库

使用 use 命令选择数据库：

USE 数据库名;

选择数据库后，你的后续 SQL 查询和操作在指定的数据库上执行。

还可以这样选择：

mysql -u 用户名 -p -D 数据库名
  • -u 用户名： 指定要连接的数据库用户名。
  • -p： 提示输入密码。
  • -D 数据库名： 指定要连接的数据库名称。
