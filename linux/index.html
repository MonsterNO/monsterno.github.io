操作系统概述

我们所熟知的计算机是由：硬件和软件所组成。

硬件： 计算机系统中由电子，机械和光电元件等组成的各种物理装置的总称。

软件： 是用户和计算机硬件之间的接口和桥梁，用户通过软件与计算机进行交流。

而操作系统，就是软件的一类。

操作系统的作用：

  1. 管理计算机硬件资源
  2. 提供用户和计算机硬件之间的接口
  3. 控制计算机的运行

常见的操作系统

  1. Windows
  2. macOS
  3. Linux
  4. Android
  5. iOS

初识 Linux

Linux 由林纳斯·托瓦兹（Linus Torvalds）在 1991 年创建，是一种自由和开放源代码的操作系统。

了解 Linux

  • Linux 是一种自由和开放源代码的操作系统，它的主要作用是管理计算机的资源和提供用户界面。
  • Linux 的一个特点是用户友好，操作简单。
  • Linux 的另一个特点是自由和开放源代码，这意味着用户可以自由地使用、修改和分发 Linux 操作系统的源代码。

Linux 内核

Linux 系统组成部分：

  • 内核：Linux 内核是操作系统的核心，负责管理计算机的硬件资源和提供用户界面。
  • 应用程序：Linux 应用程序是用户使用 Linux 操作系统的工具和程序。
  • 库：Linux 库是应用程序运行所需的代码和数据。
  • 工具：Linux 工具是用于管理和配置 Linux 操作系统的程序。

常见的 Linux 发行版

  • Ubuntu
  • Debian
  • Fedora
  • CentOS
  • Arch Linux

虚拟机

虚拟机是一种软件，借助虚拟化技术，在一台物理计算机上模拟出多台虚拟机，每个虚拟机都可以运行一个独立的操作系统。

这里使用的是 VMware Workstation 虚拟机软件。

  • 安装 VMware Workstation
  • 安装 Linux 发行版

连接 Linux

操作系统的使用形式：

  • 命令行界面（CLI）：使用操作系统提供的各类命令，以获得字符反馈的形式去使用操作系统；
  • 图形用户界面（GUI）：使用操作系统提供的可视化界面，以鼠标和键盘等输入设备去操作操作系统。

在使用 Linux 操作系统，多数使用的是命令行界面（CLI）。

这是因为：

  • Linux 从诞生至今，在图形化页面的优化上，并未重点发力。所以 Linux 操作系统的图形化页面：不好用、不稳定。
  • 在开发中，使用命令行形式，效率更高，更加直观，并且资源占用低，程序运行更稳定。

Xshell

我们使用 VMware 可以得到 Linux 虚拟机，但是在 VMware 中操作 Linux 的命令行页面不太方便，主要是：

  • 内容的复制、粘贴跨越 VMware 不方便
  • 文件的上传、下载跨越 VMware 不方便
  • 也就是和 Linux 系统的各类交互，跨越 VMware 不方便

我们可以通过第三方软件，Xshell，远程连接到 Linux 操作系统之上。并通过 Xshell 去操作 Linux 系统。这样各类操作都会十分的方便。

Xshell 是一个基于 SSH 协议的远程登录工具，它可以在 Windows、macOS 和 Linux 等操作系统上运行。

打开虚拟机，输入 ipconfig 命令，查看虚拟机的 IP 地址。

虚拟机快照

虚拟机快照是指在虚拟机运行时，对虚拟机的状态进行保存。

作用

  • 可以保存虚拟机的当前状态，以便在需要时恢复到该状态。
  • 可以在虚拟机运行时，对虚拟机的状态进行备份，防止数据丢失。
