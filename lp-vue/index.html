lp-vue

约 410 字大约 1 分钟

2024-09-05

lp-vue 是一个开箱即用的组件库，包含功能：组件&二次开发、指令、常用函数等，它可以快速的实现需求，希望能够帮助到你。

依赖

  • Vue2

  • Element

  • Echarts

安装

npm install lp-vue -S

CDN

目前可以通过 unpkg.com/lp-vue 获取到最新版本的资源，在页面上引入 js 文件即可开始使用。

<!-- 引入Vue2（必须，可自己引入其他可以访问版本） -->
<script src="https://cdn.jsdelivr.net/npm/vue@2"></script>
<!-- 引入组件库 -->
<script src="https://unpkg.com/lp-vue/lib/index.js"></script>
<!-- 使用 -->
Vue.use(LPVUE)

演示

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css" />
    <!-- 引入Vue2（必须，可自己引入其他可以访问版本） -->
    <script src="https://cdn.jsdelivr.net/npm/vue@2"></script>
    <!-- 引入组件库 -->
    <script src="https://unpkg.com/lp-vue/lib/index.js"></script>
    <title>Document</title>
  </head>
  <body>
    <div id="app">
      <lp-date-picker
        v-model="time"
        type="week"
        placeholder="请选择周"
        value-format="yyyy-MM-dd"
        format="yyyy年WW周" />
    </div>
    <script>
      new Vue({
        el: "#app",
        data: {
          time: "",
        },
      });
    </script>
  </body>
</html>

快速上手

import lpVue from "lp-vue";
Vue.use(lpVue);

按需引用

import { DatePicker } from "lp-vue";
Vue.component(DatePicker.name, DatePicker);
/* 或写为
 * Vue.use(DatePicker)
 */
