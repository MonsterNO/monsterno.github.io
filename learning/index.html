通信方式stdio

约 236 字小于 1 分钟

2025-11-10

stdio：标准输入输出

输出(stdout)

process.stdout.write("hello world");
node test.js

逻辑： 打开客户端相当于开启了父进程，使用 node 执行上诉代码，相当于开启了一个子进程，子进程会通过标准输出接口输出，父进程通过监听子进程的输出，获取子进程的输出内容，从而实现通信。

输入(stdin)

process.stdin.on("data", (data) => {
  process.stdout.write(data);
});
node test.js

逻辑： 打开客户端相当于开启了父进程，使用 node 执行上诉代码，相当于开启了一个子进程， 在父进程中输入内容，子进程通过监听标准输入接口获取父进程输入的内容，从而实现通信。

特点

stdio 通信高效、简洁，但仅适用于本地进程通信
