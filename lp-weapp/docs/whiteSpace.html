# whiteSpace 四周留白

布局组件，四周留白

# 引入

在 app.json 或 index.json 中引入组件

"usingComponents": {
   "lp-white-space": "lp-weapp/white-space/index"
}

# 代码演示

# 基础使用

<lp-white-space>
  <view>四周留白</view>
</lp-white-space>

<lp-white-space size="small">
  <view>四周留白</view>
</lp-white-space>

<lp-white-space bodyStyle="{{[20]}}">
  <view>四周留白</view>
</lp-white-space>

# API

参数 说明 类型 默认值 备注
size 留白间距模式 String defalut 可选值：
small：10rpx
default：20rpx
large：30rpx
bodyStyle 自定义边距 Array [] 数组中接受四个值，按照css逻辑实现

bodyStyle数组中数值解释

一个值：四周边距一致；

两个值：第一个值代表上下边距，第二个值代表左右边距；

三个值：第一个值代表上边距，第二个值代表左右边距，第三个值代表下边距；

四个值：按照上右下左的顺序；

同css中margin的逻辑一致。
